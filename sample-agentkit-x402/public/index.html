仕組みの概要

x402 は HTTP 402 ステータスコードを使ったオープンな支払いプロトコルです。
AgentKit は World ID と連携し、エージェントが「人間に裏付けられている」ことを証明することで free / free-trial / discount モードの優待アクセスを得られる仕組みです。

エージェントは 402 レスポンス内の extensions.agentkit.info からチャレンジを取得し、 EIP-4361 (SIWE) 形式のメッセージに署名して agentkit ヘッダーとして送信します。

本番環境では npx @worldcoin/agentkit-cli register <address> でウォレットを World ID に紐付けた後、SKIP_AGENT_BOOK 環境変数を外すと AgentBook のオンチェーン検証が有効になります。
